tomcat安装

发表于 2021-03-01 更新于 2024-06-22 分类于 tomcat

在windows上安装绿色版tomcat8

  1. 进入tomcat8解压目录
  2. 在地址栏输入cmd
  3. cd bin
  4. 安装Tomcat服务 service.bat install
  5. 启动Tomcat服务 startup.bat
  6. http://localhost:8080
