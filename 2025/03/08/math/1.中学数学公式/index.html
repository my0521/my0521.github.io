小学数学与初中数学公式大全

发表于 2025-03-08 更新于 2025-03-23

​一、数与运算

​小学部分

  1. ​四则运算

    • 加法交换律：a + b = b + a
    • 乘法分配律：a × (b + c) = a × b + a × c
    • 除法的性质：a ÷ b ÷ c = a ÷ (b × c)

  2. ​分数与小数

    • 分数乘法：(a/b) × (c/d) = (a × c)/(b × d)
    • 百分数转换：百分数 = 小数 × 100%

​初中部分

  1. ​整式运算
    • 平方差公式：(a + b)(a - b) = a² - b²
    • 完全平方公式：(a ± b)² = a² ± 2ab + b²

​二、几何

​小学部分

  1. ​周长与面积

    • 长方形面积：S = 长 × 宽
    • 圆的周长：C = 2πr

  2. ​体积

    • 圆柱体积：V = πr²h

​初中部分

  1. ​几何定理

    • 勾股定理：a² + b² = c²（直角三角形）
    • 三角形中位线定理：中位线平行于第三边且长度为其一半

  2. ​坐标系

    • 两点间距离公式：d = √[(x₂-x₁)² + (y₂-y₁)²]

​三、函数（初中）​

  1. ​一次函数

    • 表达式：y = kx + b
    • 斜率公式：k = (y₂ - y₁)/(x₂ - x₁)

  2. ​二次函数

    • 顶点式：y = a(x - h)² + k，顶点坐标(h, k)

​四、统计与概率

​小学部分

  1. ​基础统计
    • 平均数：平均数 = 总和 / 数量

​初中部分

  1. ​概率计算
    • 独立事件概率：P(A∩B) = P(A) × P(B)

​五、单位换算

​小学部分

  1. ​长度

    • 1千米 = 1000米，1米 = 100厘米

  2. ​面积

    • 1公顷 = 10000平方米

​六、三角函数（初中）​

  1. ​基本定义
    • 正弦：sinθ = 对边 / 斜边
    • 余弦：cosθ = 邻边 / 斜边

说明

  • 公式中变量含义：a, b, c为长度或常数，r为半径，h为高，k为斜率。
