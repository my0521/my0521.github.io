SpringBoot-SwaggerUi

发表于 2019-05-06 更新于 2024-06-22 分类于 SpringBoot

SwaggerUi就是自动生成接口文档的这么一个类似于插件的工具，可以直接进行接口调试

导入依赖

  1. 定义版本号
  2. 添加POM依赖

配置SwaggerConfig

注解展示接口

  1. 在接口类上添加@Api
  2. 在具体的接口上添加@ApiOperation
  • value 描述接口的具体功能
  • httpMethod 调用接口的方式

说明

在启动类上新增@EnableSwagger2注解，在controller中可以不用@Api与@ApiOperation(value = “添加一个User”,httpMethod =”POST”)

调试

  1. 启动项目
  2. 进入http://localhost:8080/swagger-ui.html
