JAVA IDE lombok插件安装

发表于 2019-05-06 更新于 2024-06-22 分类于 lombok

IDE中安装lombok插件方法

STS 安装

  1. 进入mvn中依赖lombok目录
  2. 进入cmd环境，切换到lombok目录，运行lombok-1.18.6.jar
  3. 找到STS的路径，安装lombok
  4. 编辑STS配置文件
  • 第3步成功后，在STS目录会有一个lombok.jar
  • 在SpringToolSuite4.ini最后一行添加-vmargs -javaagent:lombok.jar
